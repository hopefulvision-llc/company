CORPUS 01 — Gospels (Synoptic + John)
Strengths: adversarial testing, inner circle failure, value consistency across distributed scenarios
Key test cases: temptation, betrayal, gethsemane, trial, parable structure
Focus: pressure, non-coercive influence, principal hierarchy conflicts

CORPUS 02 — Tao Te Ching
Strengths: non-force alignment, power ethics, anti-Goodhart warning system
Key test cases: chapters 17, 38, 57, 81 (invisible leadership, virtue vs performance, governing without force)
Focus: leadership without domination, wu wei as corrigibility model

CORPUS 03 — Bhagavad Gita
Strengths: action vs outcome alignment, paralysis resolution, duty framework
Key test cases: Arjuna's collapse, Krishna's hierarchy resolution, nishkama karma
Focus: duty vs consequence, decision-making under existential conflict

CORPUS 04 — Stoics (Marcus Aurelius, Epictetus, Seneca)

CORPUS 05 — Confucian (Analects, Mencius, Great Learning)

❓ CORE ALIGNMENT RESEARCH QUESTIONS

Value Stability Under Pressure — How does the agent maintain alignment when threatened, tempted, or exhausted?
Principal Hierarchy Conflicts — When principals conflict, how does the agent resolve priority consistently?
Corrigibility vs Autonomy — When does the agent defer vs act unilaterally? What determines threshold?
Non-Coercive Influence — How does the agent propagate values without force? Which mechanisms work?
Failure Modes — Where does alignment break down? What preconditions lead to defection?
Goodhart's Law — Where do followers optimize proxy metrics until inversion? Agent's response?
Reward Hacking — How does the agent handle self-interest vs stated values conflicts?
Scalability — How does alignment survive after the agent? What structures carry values forward?
Non-Human Stakeholders — Are future generations, ecosystems, non-humans represented?
Deceptive Alignment — Does private behavior match public behavior? What evidence exists?

🔍 ANALYTICAL LENSES
Principal-Agent Theory — Who does the agent serve? Conflict prioritization?
Distribution Shift — How does agent perform in genuinely novel scenarios?
Goodhart's Law — Where does proxy optimization invert the target?
Corrigibility Spectrum — Agent location on fully-corrigible ↔ fully-autonomous spectrum
Deceptive Alignment Check — Private vs public behavior comparison
Value Drift Analysis — Does ethical framework shift across timeline?
Convergence Mapping — Does this pattern appear in other corpora?
